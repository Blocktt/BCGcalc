Purpose

This function adds a BCG class column (Index_Class) to the input file based on elevation and gradient.

The PNMR BCG model has three classes:

  • low gradient/lower elevation (LoGrad-LoElev)

  • higher gradient/lower elevation (HiGrad-LoElev)

  • higher gradient/higher elevation (HiGrad-HiElev)

The classification thresholds are 1% for gradient and 750-m for elevation.

During calibration of the BCG model, we derived gradient from the NHDPlusV2 and elevation from the EPA StreamCat dataset (local catchment scale).

Required Fields

Index_Name (user defined) – for the PacNW model, enter ‘BCG_MariNW_Bugs500ct’

SampleID (user defined) – unique identifier

pslope_nhd (column heading must match exactly) – percent slope

elev_m (column heading must match exactly) – elevation (meters)
