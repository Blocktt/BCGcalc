A single zip file will be available for download. It will include all result files along the the original data file. All result data files will be CSV format.

The zip file will included the base name of the original file along with the date and time.

The individual files will include the same basename along with a suffix to represent each step of the calculation process.

As an example if the input file was “Data_BCG_PacNW.csv” the result files will look like those in the table below.

Taxa Translator zip file contents.
File Names File Descriptions
Data_BCG_PacNW.csv User input data
Data_BCG_PacNW_taxatrans_0fileversions.csv File name versions
Data_BCG_PacNW_taxatrans_1official.csv Official taxa list
Data_BCG_PacNW_taxatrans_1attributes.csv Official taxa attributes (only if separate from official list)
Data_BCG_PacNW_taxatrans_1metadata.csv Metadata for Official taxa list
Data_BCG_PacNW_taxatrans_2taxatrans.csv Crosswalk taxa table (translations), input to official
Data_BCG_PacNW_taxatrans_3nonmatch.csv Non-matching taxa
Data_BCG_PacNW_taxatrans_MERGED.csv Merged data results
