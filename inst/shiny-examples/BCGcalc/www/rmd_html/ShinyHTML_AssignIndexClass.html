Purpose

Assign Index Class when have the input parameters in data file using BioMonTools::assign_IndexClass.

Index Name and Class Matrix

The file used to assign INDEX_CLASS by INDEX_NAME is on GitHub as part of the BioMonTools package.

File Inputs

Any file can be run through the file builder but it is suggested the file be ready for use in BioMonTools or BCGcalc functions. That is, it has the columns:

  • SITEID

    • Unique site (not sample) identifier.

  • INDEX_NAME

    • Index name used to identify the proper index

  • INDEX_CLASS

    • Column for the new class assignment.
      The existing field will be overwritten.

  • Other Optional Columns

    • Other data (e.g., date, latitude, longitude, elevation, gradient) that may be used to classify SITEID by INDEX_CLASS for the provided INDEX_NAME.

The user will be required to identify these fields in their file in the select boxes. The select boxes will contain the columns names of the user file that was already uploaded.
