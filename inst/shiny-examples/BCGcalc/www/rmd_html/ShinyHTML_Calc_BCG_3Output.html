A single zip file will be available for download. It will include all result files along wiht the original data file. All result data files will be CSV format.

The zip file will included the base name of the original file along with the date and time.

The individual files will include the same basename along with a suffix to represent each step of the calculation process.

As an example if the input file was “Data_BCG_PacNW.csv” the result files will look like those in the table below.

BCG Calculation zip file contents.
File Names File Descriptions
Data_BCG_PacNW.csv User input data
Data_BCG_PacNW_bcgcalc_1markexcl.csv Exclude taxa revised, if checked (BioMonTools::markExcluded)
Data_BCG_PacNW_bcgcalc_2metval_all.csv Metric values, all (BioMonTools::metric.values)
Data_BCG_PacNW_bcgcalc_2metval_BCG.csv Metric values, only those used in index and flags
Data_BCG_PacNW_bcgcalc_3metmemb.csv Metric membership (BCGcalc::BCG.Metric.Membership)
Data_BCG_PacNW_bcgcalc_3metrules.csv BCG rules
Data_BCG_PacNW_bcgcalc_4levmemb.csv Level membership (BCGcalc::BCG.Level.Membership)
Data_BCG_PacNW_bcgcalc_5levassign.csv Level assignment (BCGcalc::BCG.Level.Assignment)
Data_BCG_PacNW_bcgcalc_6levflags.csv BCG flags summary
Data_BCG_PacNW_bcgcalc_6metflags.csv Flag evaluations with metric values
Data_BCG_PacNW_bcgcalc_RESULTS.csv Results, levels and flags
